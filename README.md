# 🚀 Interactive Developer Quiz Game

A clean, modern, and high-performance quiz application designed for aspiring developers to test their knowledge across various technical domains.

## 🌟 About This Project
This Quiz Game is a user-centric educational tool that challenges students and developers on specific industry facts. Unlike generic quizzes, it provides a tailored experience where the user selects their professional field before starting, ensuring the content is relevant to their career goals.

## 🎯 Objectives
*   **Knowledge Validation:** To help students quickly assess their understanding of core concepts in their chosen field.
*   **Specialized Learning:** To provide field-specific paths for Frontend, Backend, Full Stack, and AI Engineering.
*   **User Engagement:** To create a "friendly" learning environment with an intuitive UI and instant feedback.

## ⚡ Key Positivities & Benefits
*   **Path Selection:** Users aren't forced into random questions; they choose their specialty (Path 1 to 5).
*   **Instant Feedback:** Visual cues (Green/Red) immediately inform the user if their answer was correct.
*   **High-Quality Aesthetics:** Uses a warm, professional color palette (`#e86a33` and `#f5efe6`) that is easy on the eyes for long study sessions.
*   **Mobile Responsive:** Perfectly optimized for both Desktop and Mobile devices.
*   **Clean Architecture:** The code is separated into logical files (HTML/CSS/JS), making it easy to scale or add more questions in the future.

## 💻 Technologies Used
*   **HTML5:** For semantic structure and accessibility.
*   **CSS3:** Custom styling featuring a premium layout, transition effects, and responsive breakpoints.
*   **JavaScript (Vanilla):** For all game logic, including dynamic DOM manipulation, category-based question loading, and score tracking.

## 📂 Project Structure
*   `index.html` - The main entry point and UI structure.
*   `style.css` - The design system and layout.
*   `script.js` - The logic and 25 fact-based questions across 5 fields.
*   `README.md` - Project documentation.

## 🛠️ How to Run
Simply open the `index.html` file in any modern web browser to start the quiz!
